Frequently Asked Questions

Do I need to be in a class setting to play this game?

In the current version for upload, This game is best used in a classroom setting with someone to guide and teach you. However, you can still play without that once you have an account! And future versions will include more independent learning.

Is this game free?

It's completely free and available for download once an account is made! The goal of this game is mainly to provide people, students or adults, a means of learning basic coding fundamentals.

Is this game available for Apple Products or phones?

Currently, this game is only available on windows. Later versions will include support for handheld devices and MAC users.
